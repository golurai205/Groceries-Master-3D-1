### Game name

**Grocery Master 3D** — Triple Match Puzzle

### Tagline

> Tap. Match. Master the pile!

### Long description

Grocery Master 3D is a cozy triple-match 3D puzzle game. Tap stacked grocery
items on the wooden table, match 3 of a kind in the tray to clear them, and
chain combos for huge bonuses. Featuring 20 hand-tuned levels, daily rewards,
4 boosters (Undo, Shuffle, Magnet, Freeze) and a satisfying kawaii art style.
